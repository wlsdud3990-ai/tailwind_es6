01_tailwindcss_기본 유틸리티 클래스

Utility-first CSS라는 이름이 생소할 수 있지만, 이미 많이 사용된 라이브러리 중에는 'Bootstrap'이 이 개념을 기반으로 제작되었습니다.

Utility-first CSS는 각 class가 담당할 스타일을 미리 정의하고 필요한 class들을 조합해서 적용하는 식으로 사용합니다.

    //class(클래스명) 이름은 다음과 같은 형태로 짓습니다. 
    .{property}{side}-{size}

1. margin과 padding값 적용하기

  • margin의 약자인 'm', 방향을 가리키는 top : t, bottom : b, left : l, right : r을 사용. mt-5 : margin-top:5px값을 정의된 속성에 따라 적용한다.
  • padding : margin과 동일한 사용법을 따라 작성함. pb-3 : padding-bottom 3px값을 정의된 속성에 따라 적용한다.
  • px-2 : x축 기준으로 padding값(padding-left, padding-right)을 2px을 정의된 속성에 따라 적용한다.
  • py-5 : y축 기준으로 padding값(padding-top, padding-bottom)을 5px로 정의된 속성에 따라 적용한다.

2. Typography(font, text, leading - 글자관련 속성)

  • 글자모양 : font-family - font-sans:font-family: ui-sans-serif, system-ui, sans-serif, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Noto Color Emoji";
  • 정렬방법 : text-left(text-align:left), text-justify(text-align:justify), text-start(text-aling:start)
  • 글자색상 : text-black(color: rgb(0 0 0 / var(--tw-text-opacity, 1));)
  • 글자속성 : font-weight : font-thin, font-extralight, font-light, font-medium, font-semibold, font-bold, font-extrabold, font-black
  • 세로정렬 : align-top, align-bottom, align-sub, align-super
  • 줄바꿈 : shitespace-nowrap, break-normal

3. 글자크기 관련 서식

  • text-xs : 12px = 0.75rem
  • text-sm : 14px = 0.87rem
  • text-base : 16px = 1rem = 1em = 100%
  • text-lg : 18px = 1.125rem
  • text-xl : 20px = 1.25rem
  • text-2xl : 24px = 1.5rem
  • text-3xl : 30px = 1.875rem

4. 글자색상, 배경색상 관련 서식

  • 글자색상 : text-sky-500/100
  • 배경색상 : bg-slate-200
  • 테두리 두께, 선모양, 선색상 : border-2, border-x-2, border-y-2, border-s-2, boder-t-2, border-dotted, border-red-500

5. 간격(spacing) - margin, padding, space between

  • margin : m-0, mx-0, my-0, ms-0, me-0, mt-0, mb-0, ml-0, mr-0, m-px
  • padding : p-0, px-0, py-0, ps-0, pe-0, pt-0, pb-0, pl-0, pr-0m, p-px
  • space-x-0 : 가로방향 자식요소들 사시 공간을 추가(가로 메뉴목록)
  • space-y-0 : 세로방향 자식 요소들 사이 공간을 추가(세로 메뉴목록)

6. 크기관련(width, height, max, min) 서식

  • 가로너비 : w-0, w-px(wedth:1px;), w04(widthL1rem.16px)
  • 세로너비 : h-0, h-px(height:1px), h-4(height:1ref, 16px)
  • 최대 가로너비 : max-w-0, max-w-px(max-wdth:1px), max-w-4(max-width:1rem, 16px)
  • 최소 가로너비 : min-w-0, min-w-px(min-width:1px), min-w4(min-width:1rem, 16px)
  • 최대 세로높이 : max-h-0, max-h-px(max-heght:1px), max-h-4(max-height:1renm 16px)
  • 최소 세로높이 : min-y-0, min-h-px(min-height:1px), min-h-4(min-height:1rem, 16px)
