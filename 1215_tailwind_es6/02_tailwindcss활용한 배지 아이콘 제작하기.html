02_tailwindcss활용한 배지 아이콘 제작하기

MD의 선택 무료배송 사은품 이벤트 소득공제

뱃지 아이콘에 적용된 유틸리티 클래스 서식 정리하기

  • inline-flex : display속성의 종류
  • items-center : flex속성 자식 요소 콘텐츠 가운데 정렬
  • rounded-md : 모서리 둥글게, sm, md, lg, xl
  • bg-gray-100 : 배경색-색상명-투명도
  • px-2, py-1 : 안쪽여백, x, y축 따로 적용
  • text-xs : text-xs(글자크기 12px), text-sm(14px), text-base(16px), text-lg(18px), text-xl(20px), text-2xl(24px)
  • font-medium : 글자 굵기 설정, font-weight: 500; font-normal : font-weight:400;
  • text-gray-500 : 글자색상, 50(투명도 97.1%), 100(93.6), 200(88.5), 300(80.8)...
  • ring-1 : 테두리
  • ring-red-500/10 : 테두리 컬러/굵기
  • ring-inset : 안쪽 테두리
bg-gray-100 : 배경색-색상명-투명도
rounded-md : 모서리 둥글게 6px
px, py : 안쪽 여백, 가로(x), 세로방향(y)
text-xs : 12px
