01자바스크립트 es06문법 소개

자바스크립트는 1995년 12월에 미국의 '브렌튼 에이크' 개발자에 의해 개발된 스크립트 프로그래밍 언어이다. 2015년도에 자바스크립트 표준인 es6문법이 발표되면서 현재까지 기준으로 많이 사용되고 있음.

1. 바닐라 스크립트 = 자바스크립트

프레임워크 또는 라이브러리가 적용되지 않은 '순수한' 자바스크립트를 의미한다.

즉, jquery, angular, vue, react같은 라이브러리/프레임워크를 사용하지 않는 순수한 자바스크립트

vanilla는 ;'기본적인(basic)', '있는 그대로', '특색없는', '평범한(plain)'이란 뜻을 지님.

2. 타입스크립트(type script)

자바스크립트의 슈퍼셋인 오픈소스 프로그래밍 언어이다. 마이크로소프트(MS)에서 개발, 유지하고 있으며 '엄격한 문법'을 지원한다.

C#언어의 리드 아키텍트이자 '델파이', '터보 파스칼'의 창시자인 Anders Hejlsberg가 개발에 참여한다.

클라이언트 사이드와 서버 사이드를 위한 개발에 사용할 수 있다.

타입스크립트는 자바스크립트 엔진을 사용하면서 커다란 애플리케이션을 개발할 수 있게 설계된 언어이다.

타입스크립트에서 '자신이 원하는 타입을 정의'하고 프로그래밍을 하면 자바스크립트로 컴파일되어 실행할 수 있다.

3. ES(ECMAScript)6 학습목차

es6문법에서 주로 많이 사용한느 문법에 대해 학습하기

  1. 변수 선언 시 키워드 - var(es6 이전 문법), let, const
  2. function(함수), 화살표 함수 =>
  3. 템플릿 리터럴(``) = 백틱문자 `{변수값}` '텍스트'
  4. class문법
  5. 비구조할당 문법
  6. 배열객체_스프레드 연산자(...)
  7. 시간객체(setInterval, timeout)
  8. 파일 첨부리스트
  9. import(가져오기), export(내보내기)
  10. Asynchronous(비동기방식)
  11. Promise(비동기방식)
